pySD.sdmout_src.ensembzarr.check_dataset_for_ensemb(dataset, refset)[source]#

returns xarray dataset after checking that time is consistent with the refset

pySD.sdmout_src.ensembzarr.ensemble_data(func, datasets, var)[source]#

call func on ensemble of var from datasets. Note: if var has dimensions [a,b, …] then data4ensemb passed into func has dimensions [n, a, b, …] when n is the number of datasets in the ensemble

pySD.sdmout_src.ensembzarr.write_ensemb_setupfile(ensembsetupfile, setupfile, datasets)[source]#

copy setupfile to ensembsetupfile and then edit / add information relevant to datasets used to make ensemble

pySD.sdmout_src.ensembzarr.write_ensemb_zarr(ensembdataset, vars4ensemb, datasets)[source]#

create zarr storage called ‘ensembdataset’ for an ensemble of zarr datasets ‘datasets’ containing their time data alongside their mean and standard deviation for the variables listed in ‘vars4ensemb’

pySD.sdmout_src.ensembzarr.write_ensemble_info(ensembsetupfile, setupfile, datasets)[source]#

add extra information in ensembsetupfile about which datasets and setuptxt file were used to make ensemble

pySD.sdmout_src.ensembzarr.write_matchingarray_to_storage(arrayname, arr, refz, zattrs)[source]#

write array ‘arr’ to zarr storage under ‘arrayname’ using same metadata as refz and copying zattrs to .zattrs file

pySD.sdmout_src.ensembzarr.write_meanvars_to_ensembzarr(ensembdataset, vars4ensemb, datasets, refset)[source]#

write mean over ensemble of datasets for each var in vars4ensemb into zarr array also called ‘var’ in ensembdataset

pySD.sdmout_src.ensembzarr.write_stdvars_to_ensembzarr(ensembdataset, vars4ensemb, datasets, refset)[source]#

write standard deviation over ensemble of datasets for each var in vars4ensemb into zarr array also called ‘var’ in ensembdataset

pySD.sdmout_src.ensembzarr.write_time_to_ensembzarr(ensembdataset, dataset)[source]#

create or replace time group in ensembdataset zarr storage with time group copied from dataset
